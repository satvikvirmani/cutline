📋 DEPLOYMENT STEPS (15 minutes total)

Step 1: Push to GitHub
  $ cd /Users/satvik/Documents/fit-tracker
  $ git push -u origin main
  GitHub URL: https://github.com/satvikvirmani/cutline

Step 2: Deploy to Vercel
  1. Go to https://vercel.com/dashboard
  2. Click "Add New" → "Project"
  3. Click "Import Git Repository"
  4. Select GitHub → Search "cutline" → Import
  5. In Environment Variables, add: APP_PASSCODE = <your-passcode>
  6. Go to Storage tab → Create Database → KV
  7. Click "Deploy"

Step 3: Test
  ✓ Visit deployed URL → Should redirect to /login
  ✓ Enter passcode → Should load tracker
  ✓ Go offline → Should still work
  ✓ On mobile → Should offer PWA install
